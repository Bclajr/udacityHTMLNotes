World Wide Web

The web is constructed of 4 different components:

  1. The Browser is a computer application that connects to the internet. There are multiple types of browsers that can be chosen from.
  2. The internet is network of computers connected to each other that passes information to each other.
  3. HTTP stands for Hyper Text Transfer Protocol and it is the protocol that the browser communicates to the servers with.
  4. Servers are computers that are designed to store information which websites are built upon.

HTML

HTML stands for Hypertext Markup Language. It is the skeleton of most web pages and other languages structure themselves around it.

HTML is made up 4 things:

  • Text content: It is what you see.
  • Markup: It is what the the content looks like
  • References to other documents: Like images and videos
  • Links to other pages: Which is anchors tags or hyperlinks that sends you somewhere else
